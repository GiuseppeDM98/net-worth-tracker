# Shared Agent Contract

Claude Code and Codex must follow the same project rules.

- Treat `AGENTS.md` as the canonical operational rulebook for implementation work, tests, localization, git hygiene, and recurring pitfalls.
- Treat `CLAUDE.md` as the architecture and current-product-status snapshot.
- When a change alters project conventions, update `AGENTS.md`; when it alters current architecture or active features, update `CLAUDE.md`.
- If `AGENTS.md` and `CLAUDE.md` conflict, prefer `AGENTS.md` for how to work and `CLAUDE.md` for what currently exists.
- Keep user-facing UI text in Italian and code/comments in English.
